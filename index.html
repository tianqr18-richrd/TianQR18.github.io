Three-level pulse-gate view

Each level places temperature immediately above the matching S11 magnitude and phase charts. Temperature is always shown as absolute °C. S11 magnitude and phase use focused change-from-reference scales; tap or hover to read their absolute values. In Level 1, two vertical guides bound the selected 1 s window. In Level 2, the vertical reference marks the selected pulse. Level 3 contains only the retained HIGH-gate samples from that pulse.

Level 1 — global run with selected-second window

One RF HIGH pulse-gate aggregate per second, aligned with temperature. The x-axis is centred on the selected second so the two guides always show its exact 1 s boundary.

Level 2 — selected 1 second with selected-pulse marker

One RF HIGH pulse-gate estimate per valid 1 ms pulse, aligned with temperature. Zero on the x-axis is the pulse selected for Level 3.

Level 3 — selected 1 ms pulse

Samplewise S11 inside the edge-trimmed RF HIGH gate. Temperature is constant over this microsecond-scale window, so the absolute selected-pulse temperature is shown as a card rather than a misleading flat axis.

Method and downloads

This page implements the computation used by TMTT_v1_9_121347_S11_three_level_pulse_only.ipynb without publishing the notebook itself. RF OFF, out-of-pulse samples, low-power gates, low-coherence gates, and insufficient-width gates are excluded. The page is a published snapshot, not a live instrument connection.
