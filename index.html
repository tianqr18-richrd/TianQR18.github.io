Three-level pulse-gate view

Each level places temperature immediately above the matching S11 magnitude and phase charts. All x-axes within a level use the same time basis.

Level 1 — global run

One RF HIGH pulse-gate aggregate per second, aligned with temperature.

Level 2 — selected 1 second

One RF HIGH pulse-gate estimate per valid 1 ms pulse, aligned with temperature.

Level 3 — selected 1 ms pulse

Samplewise S11 inside the edge-trimmed RF HIGH gate; temperature is shown on the same microsecond axis.

Method and downloads

This page implements the computation used by TMTT_v1_9_121347_S11_three_level_pulse_only.ipynb without publishing the notebook itself. RF OFF, out-of-pulse samples, low-power gates, low-coherence gates, and insufficient-width gates are excluded. The page is a published snapshot, not a live instrument connection.
